## How to Use This Tracker

1. **Claiming a task:** Change status to `IN PROGRESS` and add your name to Owner
2. **Completing a task:** Change status to `DONE` and add completion date
3. **Blocked?** Change status to `BLOCKED` and note what's blocking you
4. **After making changes:** Save and commit:
   ```
   git add .
   git commit -m "Update: [brief description of what changed]"
   ```
